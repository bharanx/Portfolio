NETRA – Portable Network Security Device

Hardware-Based Inline IDS/IPS Gateway & Real-Time Telemetry Dashboard

Problem: Public Wi-Fi networks expose users to Man-in-the-Middle (MITM) hijacking and scanning, while client-side host configurations are often weak or disabled.

Solution: Developed a plug-and-play network security appliance that sits as a transparent gateway between client hosts and untrusted routers, actively intercepting and filtering transit network packets.

Engineering & Complexity:

  • Configured a physical Raspberry Pi as a routed wireless access point to act as a hardware traffic proxy.
  • Routed interface packets transparently through a Suricata IDS engine with custom rules to inspect signatures.
  • Created an automated mitigation daemon that dynamically appends blocking rules to kernel-level iptables on threat detection.
  • Built a WebSocket alert server in Node.js to stream live security telemetry to an administrative React panel.
